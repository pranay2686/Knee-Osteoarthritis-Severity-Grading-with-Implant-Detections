# Knee-Osteoarthritis-Severity-Grading-with-Implant-Detections
Deep learning pipeline for automated Kellgren-Lawrence grading of knee osteoarthritis in plain radiographs, including post-surgical patients with metallic implants. Features MobileNetV2 implant detection, metal artifact suppression, and a DenseNet121+ConvNeXtTiny weighted ensemble with Grad-CAM explainability.
